# Alarmy Clocky app

A simple alarm app that lets users record custom audio clips and use them as alarm sounds. Includes audio recording, playback, persistent storage and an alarm trigger.

To be developed: 
- UX and UI improvements
- sending your alarms to friends 

Technologies used: 
- React Native (UI and app logic, components, hooks, styling)
- Expo (development environment, tooling and builds)
- Expo AV (audio recording and playback APIs)
- Typescript (typed superset of JavaScript)
- JavaScript (time handling, string manipulation, array operations)
- AsyncStorage (storage solution)
- Git + Github
